#Aprender classes Wrappers

## São classes que já possuem a implementação necessaria para realizar cast de tipos primitivos

> Para quase todos os tipos primitivos Existe uma classe com o mesmo nome, as iniciando com letra Maiuscula
> Essas classes possuem métodos estáticos que recebem por argumento Strings e converte no tipo primitivo respectivo

> - Exceto char que possui a classe Charactere e int que possui a classe Integer
